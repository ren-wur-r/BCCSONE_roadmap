## 5. 報告系統詳細說明 (BCCS-Generative-Report-System)

> Repo 位置: `BCCS-Generative-Report-System-master/`
> 技術架構: React 18 + FastAPI + SQL Server + Playwright
> 系統版本: v1.78.180 (stable)

報告功能並非在 SOC-Portal 本體實作，而是作為獨立系統 **BCCS-Generative-Report-System** 開發，透過 SOC 資料庫連線取得資料後產出報告。

### 5.1 報告系統功能矩陣

| 功能                 | 狀態   | 說明                                                                               |
| -------------------- | ------ | ---------------------------------------------------------------------------------- |
| **合規模版管理**     | 已完成 | 預建標準模版、部門/標籤權限控制、版本管理、編輯歷史、模版鎖定(防並行衝突)          |
| **客製化報告編輯器** | 已完成 | 拖曳式 Canvas 編輯器，支援 7 種元素(標題/段落/表格/圖表/圖片/時間區間/分隔線)      |
| **動態資料綁定**     | 已完成 | SQL 查詢、排程檔案、內嵌資料、AI 結論 等多種資料來源                               |
| **PDF 匯出**         | 已完成 | Playwright 渲染引擎(高擬真度)，附 ReportLab 備援；支援中文字型、自動分頁、目錄生成 |
| **排程自動產出**     | 已完成 | Cron 排程定義、自動執行、執行紀錄追蹤                                              |
| **Email 寄送**       | 已完成 | SMTP 設定管理、聯絡人管理、排程報告自動寄送                                        |
| **AI 智慧結論**      | 已完成 | Google Gemini / Anthropic Claude 整合，自動產生報告摘要                            |
| **權限控制**         | 已完成 | super_admin / internal / external 三層角色，模版可視範圍控制                       |
| **稽核日誌**         | 已完成 | 完整操作追蹤，JSON Diff 變更檢視                                                   |
| **SOC 資料庫整合**   | 已完成 | 連線 SOC DB 查詢警報/事件資料，支援 SQL Auth 與 Windows Auth                       |
| **Excel/Word 匯出**  | 未實作 | 目前僅支援 PDF 格式                                                                |

### 5.2 報告系統與 SOC 戰情中心的整合方式

```
SOC 戰情中心 (SOC-Portal)          BCCS-Generative-Report-System
┌─────────────────────┐           ┌──────────────────────────────┐
│  PostgreSQL          │           │  SQL Server                  │
│  - alerts            │           │  (AutoReportSystem)          │
│  - incidents         │           │  - rpt_templates             │
│  - users             │           │  - rpt_report_records        │
│  - organizations     │           │  - rpt_schedule_configs      │
└────────┬────────────┘           │  - ...30+ tables             │
         │                        └──────────┬───────────────────┘
         │  SOC DB 連線                      │
         │  (SQL Server/mssql)               │
         └──────────────────────────►────────┘
                                   ↓
                           Report System 查詢 SOC 資料
                           產出 PDF 報告
```

- 報告系統透過 **SOC 資料庫連線** 直接查詢 SOC 的警報、事件等資料
- 使用 `SESSION_CONTEXT` 實作列級安全性(Row-Level Security)
- 支援專案/公司維度的資料篩選
- 兩個系統各自獨立部署，透過資料庫層整合

## 6. 三大平台通用模組架構建議

根據圖片架構，以下功能應設計為跨平台通用模組，供 SOC 戰情中心、弱點管理大平台、資安大師 BCCS One 共用：

| 通用模組         | 說明                                               | 共用方式           |
| ---------------- | -------------------------------------------------- | ------------------ |
| **儀表板引擎**   | 可配置的 Widget 系統、拖曳布局、資料來源抽象       | 共用元件庫         |
| **工單案件管理** | Alert/Incident 生命週期、指派、稽核、附件          | 共用 API + UI 元件 |
| **報告引擎**     | 合規模版管理、客製化報告、排程產出、PDF/Excel 匯出 | 共用服務模組       |
| **風險評鑑計算** | 風險評分模型、量化公式、趨勢追蹤                   | 共用計算引擎       |
| **人員組織**     | 組織架構、角色權限、多租戶隔離                     | 共用認證/授權服務  |
