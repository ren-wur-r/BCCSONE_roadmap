## 4. 與其他平台的整合分析

### 4.1 與弱點管理大平台的整合（Roadmap 標示）

Roadmap 圖中，弱點管理大平台向資安大師 BCCS One 有資料流連接，共享以下模組：

| 共享資料/功能 | 弱點管理大平台側                             | 資安大師側                     | 整合現況 |
| ------------- | -------------------------------------------- | ------------------------------ | -------- |
| **資產清冊**  | 弱掃目標資產清冊                             | 組織資產管理 + CIA 分級        | 未整合   |
| **人員組織**  | 弱掃負責人指派                               | 部門/使用者/角色 RBAC          | 未整合   |
| **風險評鑑**  | 弱點風險評分                                 | ISO 27001 風險評鑑流程         | 未整合   |
| **弱點資料**  | 弱掃工具（Acunetix/Nessus 等）+ 源碼檢測結果 | 弱點管理模組（匯入/追蹤/修補） | 未整合   |

### 4.2 與 SOC 戰情中心的整合

Roadmap 圖中 SOC 戰情中心與資安大師並無直接資料流，但兩者皆連接至弱點管理大平台的風險評鑑。

| 共享概念         | SOC 戰情中心                   | 資安大師                      | 整合潛力           |
| ---------------- | ------------------------------ | ----------------------------- | ------------------ |
| **工單案件管理** | Alert/Incident（安全事件導向） | Incident/Vuln（合規治理導向） | 可共用工單引擎     |
| **風險評鑑計算** | 安全事件風險評分               | ISO 27001 風險評鑑            | 可共用計算引擎     |
| **報告引擎**     | Generative-Report-System       | 尚未對接                      | 可直接對接         |
| **儀表板**       | 安全監控儀表板                 | 合規治理儀表板                | 可共用 Widget 引擎 |
